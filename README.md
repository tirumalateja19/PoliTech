## 📌 Overview

PoliTech Solutions is a modern political consulting and campaign support platform designed to present strategic services in a clean and professional format. The website focuses on campaign strategy, voter engagement, communication support, and digital outreach through a responsive and visually polished user experience.

The project emphasizes minimal design, structured layouts, smooth navigation, and brand-focused UI interactions built with React and Tailwind CSS.

---

## ✨ Features

- Responsive navigation bar with mobile menu
- Animated hero section with rotating text
- Interactive service cards
- Brand-aligned color palette and layout
- Professional contact footer with inquiry form UI
- Smooth scroll navigation
- Responsive design across devices
- Clean and scalable component structure
